# 📚 Student Study Planner

This is a **web-based study planner** that helps students manage their study tasks and deadlines.  
The application allows users to add, edit, delete, and track tasks easily.

## Features

- User login and signup
- Add, edit, and delete tasks
- Task priority system
- Subject categories
- Search and filter tasks
- Task statistics dashboard
- Progress bar
- Productivity pie chart
- Deadline reminder for tasks due today
- Dark mode

## Technologies Used

- HTML
- CSS
- JavaScript
- Chart.js
- LocalStorage

## Purpose
The goal of this project is to help students organize their study tasks, manage deadlines, and track their productivity in a simple way.
